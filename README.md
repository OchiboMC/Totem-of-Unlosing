# Totem of Unlosing

## 不失のトーテム
対応ver: 1.21.11  
**導入は自己責任で！大事なワールドならバックアップを取ってね！**  

### 簡単な説明
死んでしまいますが、アイテムを保持してくれるトーテムを追加するデータパックです。  
経験値も保持してくれますが、普通に死ぬ時よりも多く失ってしまいます。

### 不失のトーテムの入手方法
- 古代都市のチェストからまれに手に入ります。  
- エヴォーカーが「不死のトーテム」の代わりにまれにドロップします。

### 細かい仕様
- まっさらなトーテムを手に持って死亡するとトーテムが発動し、アイテムを保持します。
- アイテムを保護したトーテムは、手に持って右クリックで開封できます。
- 開封すると、そのトーテムが保持した時点のインベントリを再現するようにアイテムが変換されます。また、開封時に持っていたアイテムは周りにばら撒きます。
- トーテム発動時、保護済みトーテムがインベントリ内にある場合はそのトーテムは自動で開封され、新しいトーテムに保存されます。
- また、トーテムは60スロット分までアイテムを保存できますが、超過分のアイテムは死亡時にドロップされます。
